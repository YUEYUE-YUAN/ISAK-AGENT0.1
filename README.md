# Agent Date

一个基于LangGraph和LangChain的智能体应用项目。

## 🚀 功能特性

- 基于LangGraph的状态管理
- 多智能体协作
- OpenAI API集成
- 模块化设计

## 📋 环境要求

- Python 3.12+
- OpenAI API密钥

## 🛠️ 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd agent-date
   ```

2. **创建虚拟环境**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Linux/Mac
   # 或
   venv\Scripts\activate     # Windows
   ```

3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **配置环境变量**
   ```bash
   cp .env.example .env
   # 编辑 .env 文件，添加您的API密钥
   ```
