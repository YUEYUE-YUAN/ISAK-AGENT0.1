# ISAK-AGENT0.1
 🎯 基于 LangGraph 的个人智能体系统   
一个可成长的“数字副脑”，融合语音理解、任务规划、上下文记忆与行动执行。   
具备多智能体协作架构，支持私有部署、跨平台使用，构建属于自己的 AI 伴生体。
